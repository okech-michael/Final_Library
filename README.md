#  Digital Library System

A complete online library platform providing access to academic materials from **primary school, high school, and university**. Users can read, download, bookmark, subscribe for premium content, and manage their own learning resources in one central place. Built for accessibility, scalability, and efficient e-learning distribution.


## Overview

The **Digital Library System** is a full-stack web application designed to operate as a modern online library. It eliminates the need to physically visit a library by giving users authenticated access to thousands of digital materials â€” including notes, books, past papers, guides, research papers, and more.

The system supports user accounts, admin uploads, categorization, payments, and offline saving. Premium materials can be purchased individually or unlocked via subscription packages.



## Key Features

###  User Features
- User registration & login
- View and download free materials
- Purchase or subscribe for premium content
- Save books for offline access
- Add items to cart / wishlist
- Manage profile & reading history
- Search and filter materials by category, level, or subject

###  Admin Features
- Upload learning materials (PDF, Docs, Videos, etc.)
- Categorize materials (Primary, High School, University)
- Set items as Free or Premium
- Manage users
- View platform activity and downloads



## Payment Integration

Supports **M-Pesa Daraja API** for:
- One-time payments
- Subscription plans
- Automatic payment confirmation via callbacks



##  Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Bootstrap / Tailwind
- **Backend:** Django / Python
- **Database:** PostgreSQL / MySQL
- **Storage:** Local or cloud (AWS S3, Cloudinary, etc.)
- **Payments:** M-Pesa Daraja API
